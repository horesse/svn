Введение

Макеты страниц — это набор готовых макетов, которые можно использовать в качестве основы для дизайна любой страницы или приложения. Хотя по умолчанию они предоставляют некоторый стиль, он очень минимален и может быть легко изменен из созданного вами компонента.

Основные преимущества использования макетов страниц:

  • Последовательность

    Все ваши приложения и страницы будут выглядеть одинаково с точки зрения макета и общего стиля, что, по сути, сделает их более удобными для пользователя и простыми в освоении.

  • Простые модификации

    В будущем, если вы решите изменить дизайн своих страниц, добавить элементы или изменить существующие, это можно будет сделать очень легко, поскольку все ваши страницы будут иметь одинаковые имена классов и общую структуру кода.

  • Более простая кривая обучения для ваших пользователей

    Если общий дизайн и макет ваших страниц будут одинаковыми, вашим пользователям будет легче их изучать. Им не придется снова и снова искать кнопку «Сохранить» или поле поиска, пытаясь найти и запомнить их местоположение на каждой странице, поскольку все страницы будут иметь одинаковую структуру.
